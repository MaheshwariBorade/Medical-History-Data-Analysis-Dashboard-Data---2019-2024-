This project is an Excel-based dashboard developed to analyze 55,000+ patient records from 2019 to 2024. It features interactive Pivot Tables, charts, slicers, and advanced Excel formulas to provide insights into medical conditions, billing, insurance data, admission types, and test results.

## Features

- Dynamic data visualization with Pivot Tables and slicers
- Automated reporting using advanced Excel formulas (VLOOKUP, HLOOKUP, conditional formatting)
- Comprehensive analysis of patient demographics, conditions, billing, and insurance
- Easy-to-use dashboard designed for healthcare management teams

## Skills Demonstrated

- Excel dashboard creation and data cleaning
- Advanced Excel functions and formulae
- Data visualization and reporting
- Collaboration and project documentation

## Usage

Open the included Excel file to explore interactive reports and dashboards based on the comprehensive medical dataset.
